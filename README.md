# 📖 Blog <a name="about-project"></a>

A blog is full stack app where user can post an article, read, likes and comments on the article
## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

<details>
  <summary>Client and Server</summary>
  <ul>
    <li>Rails View</li>

  </ul>
</details>


### Key Features <a name="key-features"></a>

- Each user and post have unique id
- Admin console to use and monitor the blog
